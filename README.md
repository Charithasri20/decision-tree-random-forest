# Decision Tree + Random Forest

## Overview
Complete implementation of Decision Tree and
Random Forest classifiers on the Iris dataset
with visualization, feature importance analysis,
and cross-validation comparison.

## Topics Covered
- Decision Tree training and visualization
- Effect of max_depth on overfitting
- Random Forest with 100 trees
- Feature importance bar chart
- Confusion matrix
- 5-fold cross-validation comparison
- Model saving with pickle

## Technologies Used
- Python 3, NumPy, Pandas
- Matplotlib, Seaborn, Scikit-learn

## Dataset
Iris — 150 flowers, 3 species (sklearn built-in)

## Key Results
- Decision Tree Accuracy: ~97%
- Random Forest Accuracy: ~97%
- Most important features: petal length and width
- Cross-validation confirms consistent performance

## How to Run
Open in Google Colab → Runtime → Run all
